# 🚀 Sawan Kumar – Resume Website

A modern, responsive, and theme-customizable personal resume website built using **React (Vite)** and custom CSS.

This portfolio highlights my skills, projects, education, and contact information in a clean and interactive UI.

## 🚀 How to Run Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/resume-website.git
cd resume-website
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```
